tropicosMassModel
Adam B. Smith | last updated 2020-06-01

This folder contains scripts and files necessary to run a continuous series of models on specimen data. The primary file is entitled "tropicosMassModel.r". Instructions can be found the subfolder "vignette".

Folders:
\code: Functions used to run rTropicosMassModel
\support: Supporting data files needed to run rTropicosMassModel, not generated by the code
\regions: Supporting data files needed to run models, generated by the code
\models: Output
\temp: Temporary directory, sometimes created/deleted by rTropicosMassModel.r
